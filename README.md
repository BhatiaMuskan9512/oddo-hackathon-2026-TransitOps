# oddo-hackathon-2026-TransitOps

# Project Name

**TransitOps – Smart Transport Operations Platform**

---

# Problem Statement

Many organizations still rely on spreadsheets and manual records to manage transport operations. This often leads to scheduling conflicts, underutilized vehicles, missed maintenance, expired driver licenses, inaccurate expense tracking, and limited operational visibility.

TransitOps provides a centralized platform to manage the complete transport lifecycle, including vehicles, drivers, trips, maintenance, fuel logs, expenses, and operational analytics.

---

# Solution

TransitOps is a web-based transport management system that digitizes fleet operations through a centralized platform. It enables organizations to efficiently manage vehicles, drivers, trips, maintenance, fuel consumption, and expenses while enforcing business rules and providing real-time operational insights through a dashboard.

---

# Features

- User Authentication with Role-Based Access Control (RBAC)
- Vehicle Management
- Driver Management
- Trip Management
- Maintenance Management
- Fuel Log Management
- Expense Management
- Dashboard with Key Performance Indicators (KPIs)
- Reports and Analytics

# Installation

### Clone the Repository

```bash
git clone https://github.com/BhatiaMuskan9512/oddo-hackathon-2026-TransitOps.git
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
